The kernel configuration as const items.

While this module can be used as an alternative to the sel4_cfg_*! macros for accessing the kernel configuration at the value level, its primary purpose is to provide a reference within Rustdoc for the active configuration. Towards that end, the generated source of this module is also provided in this module’s Rustdoc to make browsing easier.
